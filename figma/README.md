# Figma challenge

You are sent back to the 80s, consumer electronics are very basic and computers are only used in scientific research.

You are tasked to design a control panel for a TV broadcasting station.

Using Figma, come up with a period correct control panel that can be used for mixing, splicing and broacasting Live TV.

You have full creative freedom, but it must only feature technologies available at the time


# Défi Figma

Vous êtes envoyés dans les années 80, les électroménagers sont très simples et les ordinateurs ne sont utilisés que pour des fins scientifiques.

Vous devez créer un paneau de contrôle pour une station de diffision de télévision.

Utilisez Figma pour illustrer un concept de panneau de contrôle qui être utilisé pour le "mixing", la découpe et la diffusion de télévision en direct.

Vous avez la liberté créative complète, mais le concept ne peux contenir que des technologies disponnibles à cette époque.
