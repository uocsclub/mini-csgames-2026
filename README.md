# uOCS Mini CSGames 2026

# Event format

There are going to be 5 different challenges you can submit solutions for.

You can work on any number of challenges and you are not required to finish them

You must submit your solutions by email before January 25th at 18h. The email will be given in person.

Some challenges will be on paper and will require you to turn in the solutions in person

# Challenge overview

- Theoretical  
    Answer theoretical CS questions on paper
- Digital design  
    Complete some VHDL implementations according to specified constraints
- Low level design  
    Implement in C a solution to a given problem
- Design  
    Something to do with figma
- Proofs  
    Solve some proofs with the lean proof solver

Detailed instructions for each challenge will be available in their corresponding folder

Good luck!
